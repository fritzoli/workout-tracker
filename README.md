# Workout Tracker (Work in Progress)

This repository contains the backend of a training tracking application that is currently under development. 
The focus of the project is on creating a structured and scalable backend system for managing training-related data. 
It should enable users to create training plans, track and evaluate their own workouts, as well as other fitness-related characteristics such as weight.

## Functions implemented so far
- User signup with email verification
- User login with JWT token generation
- User email verification flow (send verification link, verify user)

- Weight tracking functionality
- Creating training exercises
- Global error and exception handling with standardized format
